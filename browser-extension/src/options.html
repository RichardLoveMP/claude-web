Claude Code Web Assistant

把网页里的代码和文字，一键带到本地 Claude Code Web。

1 启动本地服务

运行 claude-web，保持 Web UI 打开。

2 生成 Token

在 Claude Web 设置里生成并复制插件 Token。

3 选中即问

网页中选中内容，右键选择 Claude Code Web。

本地连接

插件只连接 localhost，不会把 Token 发到外部服务。
